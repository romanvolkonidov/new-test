╔════════════════════════════════════════════════════════════════╗
║                    HOW TO FIX IN VERCEL                        ║
╚════════════════════════════════════════════════════════════════╝

  1. Go to: vercel.com
  
  2. Click your project
  
  3. Click: Settings
  
  4. Find: "Root Directory"
  
  5. Click: Edit
  
  6. Type: app
  
  7. Click: Save
  
  8. Go to: Deployments tab
  
  9. Click: ⋮ (three dots)
  
  10. Click: Redeploy
  
  ✅ DONE!


╔════════════════════════════════════════════════════════════════╗
║              OR USE VERCEL CLI (EASIER!)                       ║

  Open terminal and run:
  
    cd /home/roman/Documents/LiveKit/app
    npm install -g vercel
    vercel login
    vercel --prod
  
  That's it! ✅
